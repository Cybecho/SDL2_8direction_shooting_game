# SDL2_GAME_KILL_EM_ALL
SDL2라이브러리로 제작한 8방향 이동 슈팅게임

## 구현 사항
1. 총알은 최대 3발까지 발사 가능
2. 총알 재장전 기능 구현
3. 총알은 화면을 벗어나면 소멸자가 호출됨
4. 8방향 이동
5. 실행할때마다 맵 랜덤으로 변함
6. 총알 발사할때마다 10점씩 획득
7. 인트로페이즈, 게임플레이페이즈, 엔딩페이즈 구분 (우클릭으로 전환)
8. 게임플레이 페이즈는 매번 초기화됨

## AI 리소스
1. BGM : AI Genarate
